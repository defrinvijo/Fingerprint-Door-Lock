OVERVIEW:

The Fingerprint Based Smart Door Lock System is a secure access control project developed using Arduino and a fingerprint sensor. The system allows only authorized users to unlock the door using their registered fingerprints. It provides enhanced security through biometric authentication and includes additional features such as automatic locking, status indication, and security alerts for unauthorized access attempts.

FEATURES:

Fingerprint-based authentication
Secure door locking and unlocking using a servo motor
Automatic door locking after a predefined time
LCD display for system status and user feedback
Green LED indication for successful authentication
Red LED indication for failed authentication
Buzzer notifications for access granted and denied
Intruder alert after multiple failed attempts
Easy fingerprint enrollment and management
Low-cost and reliable security solution
